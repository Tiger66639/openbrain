openbrain by hallowname

An AIML Interpreting Plasmoid for the KDE4 desktop.

OpenBrain will one day understand most simple computing tasks, such as 'check email' or 'look on youtube for flying penguins'. Not only this, but more complex things can be accomplished by mixing C++ with AIML. AIML is simple to program, and most humans are familiar with standardized markup languages, therefore OpenBrain will be drastically simpler to program than any desktop assistant before it. Being open-source, free, and easy to program, OpenBrain should develop quite rapidly.

Dependencies

Qt4.svn(4.4.3 or greater), KDE4.svn(4.1.85 or greater)

Install

OpenBrain is built on the BSD licensed CMake build-system, which is cleaner and even more free than autoconf and friends. To install OpenBrain (the quickest way, via terminal): Download the tarball (.tar.bz2 file) Open a terminal, and cd to the folder containing the .tar.bz2 file type: 'tar jxf openbrain-0.0.2.2.tar.bz' followed by enter, to decompress the tarball type 'cd openbrain-0.0.2.2' to enter OpenBrain's source directory type 'cmake .' to configure the source for your system (and check for missing dependencies) type 'make' to compile (build) the source code into the finished product (OpenBrain) type 'sudo make install' to install OpenBrain into your systems execution paths (so you can use it :P) That's it... OpenBrain is installed... Just right-click your KDE4 desktop, select 'Add Widget', and add the OpenBrain plasmoid.
